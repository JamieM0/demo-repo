# TinyTasks

A tiny, dependency-free task tracker used as a demo/playground repo for exercising
[BranchKit](https://github.com/JamieM0/BranchKit).

## Running it

Open `index.html` in a browser, or run `npm start` to serve it over HTTP on port 8080.

## Project layout

- `index.html` — entry point
- `src/app.js` — app bootstrap + rendering
- `src/utils.js` — small helpers (date formatting, id generation)
- `src/styles.css` — styling
- `assets/` — logo + icons
- `data/sample-data.json` — sample seed data (also useful as a "large file" test case)
